# PromiseProof YouTube launch plan

## Best first video title

I Built a Website That Proves If YouTubers Actually Pay Winners

## Video concept

1. Create a challenge on PromiseProof.
2. Fund a small reward.
3. Hide a code in the video.
4. Ask viewers to submit the code on the proof page.
5. Pay the winner.
6. Show the public proof event: funded, rules locked, winner verified, paid.

## Safe reward ideas

- $1 tip or donation
- private Discord invite
- shoutout
- channel review on stream
- profile badge on PromiseProof
- featured creator card

Avoid making likes/subscribers mandatory. Social actions can be optional, but the real entry should happen on PromiseProof.

## Channel positioning

PromiseProof is the place where creator promises become public proof.

Slogan: Creators make promises. We prove them.
