# PromiseProof MVP roadmap

## V1 — account-first proof pages

- Supabase Auth with email and Google
- Profiles with handle, display name, avatar, role, identity status
- Social account linking page
- Payout account page for PayPal/Stripe placeholders
- Hidden-code challenges
- Public proof pages
- Promise Score calculations
- Admin verification queue
- Support tickets and message threads UI

## V2 — money flow

- Stripe Connect onboarding
- Creator deposits/funds challenge
- Winner payout flow
- Public payout receipt record
- Refund logic
- Better fraud/risk checks

## V3 — game verification

- API verified games first
- Controlled Match Room records
- Moderator review dashboard
- Leaderboard Crown Rush challenges

## V4 — trust network

- Public creator verification badges
- Creator profiles with social accounts
- Dispute history
- Moderator notes
- Better support inbox and notification emails
